7x12-fonts
==========

애플2시리즈의 고해상도 그래픽(HGR; High Resolution Graphics) 모드 280x192를 최대한 활용하기 위해 만튼 비트맵 글꼴을 트루타입으로 변환.

## 영문 7x12, 한글 14x12

영문 280/7=40열, 한글 280/14=20열, 192/12=16행

- [미리보기](https://iolo.kr/7x12-fonts/?font=7x12)

## 영문 5x12, 한글 10x12

영문 280/5=56열, 한글 280/10=28열, 192/12=16행

- [미리보기](https://iolo.kr/7x12-fonts/?font=5x12)

## 조합 규칙 3x1x1

- 초성1: 세로모음(ㅏ,ㅐ,ㅓ,ㅔ,ㅕ,ㅖ,ㅣ)
- 초성2: 가로모음(ㅗ,ㅛ,ㅜ,ㅠ,ㅡ)
- 초성3: 복모음(ㅘ,ㅙ,ㅚ,ㅝ,ㅞ,ㅟ,ㅢ)

> (영문 글자당 12bytes) * 128 + (한글 글자당 24bytes)*(초성 19자*3벌 + 중성 21자 + 종성 28자) =  4080 bytes = 4K!
